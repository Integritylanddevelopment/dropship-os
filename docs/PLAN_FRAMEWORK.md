# Dropshipping Plan Framework

## 1. Research Engine (Stock Market Model)
- Cheapest Attention Tracker (CPM/CPC by channel)
- Highest Margin Product Tracker (cost vs. sell price)
- Viral Content Tracker (what's trending, who to piggyback)
- Competitor Copy Log (winning ads, pages, hooks)

## 2. Decision Layer
- Rule: Match highest-margin product to cheapest-attention channel
- Weekly reallocation based on research data
- Kill/scale triggers (CPA, ROAS thresholds)

## 3. Asset Stack
- One-pagers hosted on GitHub Pages
- Stripe checkout integration
- Template library (reusable landing pages)
- Domain/brand rotation system

## 4. Content Machine (Gary V Model)
- Pillar content → micro-content repurposing
- Platform-native formats (TikTok, Reels, Shorts, X)
- Jab-Jab-Jab-Right-Hook cadence
- Trend-jacking / piggyback playbook

## 5. Automation Pipeline
- Auto site spin-up (template → GitHub Pages → Stripe)
- Auto content posting across channels
- Auto data pull into research dashboard

## 6. Metrics Dashboard
- CPM, CPC, CTR by channel
- Margin by product
- ROAS by combo (product × channel)
- Reallocation signals
