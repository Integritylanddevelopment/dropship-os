Agent Command Center
Run any agent function from your terminal — copy the exact command
🔧
Initial Setup
Connect APIs, build board structure, seed keywords, build seasonal calendar
python main.py setup
🔍
Research Cycle
Discover keywords, score opportunities, identify seasonal priorities
python main.py research
✍️
Generate Content
AI-generate pin titles, descriptions, lead magnets, weekly plans
python main.py content --count 20
🗂️
Create Boards
Deploy board structure to Pinterest with SEO-optimized descriptions
python main.py boards
🚀
Publish Pins
Deploy queued pins to Pinterest via API. Respects rate limits.
python main.py publish --count 15
📊
Analytics Report
Pull fresh pin/account analytics, identify top performers
python main.py analytics
📅
Seasonal Planning
Build seasonal calendar, generate time-sensitive content
python main.py seasonal
🎯
Score Opportunities
Run 10-dimension opportunity scoring to find best content areas
python main.py score
🔄
Full Growth Cycle
Run everything: research → boards → content → publish → analytics
python main.py full-cycle
👁️
Dry Run Preview
Preview all actions without making changes to Pinterest
python main.py publish --dry-run
🔑
Niche Research
Research a specific niche or product category with custom seed keywords
python main.py research --niche "kitchen gadgets"
📄
Export Analytics
Export full analytics data as JSON for spreadsheet analysis
python main.py analytics --output json

python main.py full-cycle
