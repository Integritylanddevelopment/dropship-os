Integration Hookup Guide
Fastest path to live data — in priority order
1
Connect Shopify via API key
Unlocks ~40% of all metrics: Revenue, margins, LTV, refund rate, AOV, repeat rate. Fastest ROI on setup time.
2
Connect TikTok Ads API
Unlocks cheapest-attention ranking. CPM, CTR, ROAS, creative performance. Needed to run the attention arbitrage map.
3
Connect Meta Ads API
Completes the paid channel picture. Creative fatigue scoring needs frequency data from here.
4
Connect Email Platform (Klaviyo recommended)
Unlocks owned-channel economics: revenue per email, list growth rate, sequence performance.
5
Connect Google Ads + Analytics
Shopping ROAS and organic search data. Needed to fully rank channel attention cost.
6
Connect Expense Tracking
Without this, net margin and cash flow metrics stay dark. QuickBooks, Xero, or a simple spreadsheet export works.
7
Connect TikTok + IG organic analytics
View counts, engagement, follower growth. Needed to measure content engine KPIs accurately.
8
Add manual scoring (Offer + Value Equation)
Some metrics can't be automated — offer score, hook format testing, value equation inputs. Score weekly manually.
